#### Start r2 :
```r
r2 -A ./bin             # open binary + auto analysis
r2 -d ./bin             # debug mode
r2 -w ./bin             # open in write (patch) mode
r2 -e io.va=false ./bin # treat seeks as file offsets
```
#### Help / basics :
```r
?                      # help
?e                     # examples
q                      # quit
e scr.color=true       # colors
```
#### Navigation :
```r
s                      # show current offset
s 0x401000             # seek to address
so                     # seek to previous
s+ 0x100               # forward
s- 0x100               # back
px 64                  # hexdump 64 bytes
pc 64                  # bytes as chars
```
#### Analysis :
``` r
aa                     # basic analysis
aaa                    # deep analysis (common)
af                     # analyze current function
afl                    # list functions
afi                    # function info
afn newname            # rename current function
```
#### Disasm :
```r
pd 20                  # disasm 20 instr
pD 64                  # disasm 64 bytes
pdf                    # disasm current function
pdr                    # disasm with refs
```
#### Graph View :
```r
VV                     # visual graph (space/enter to navigate)
agf                    # ascii graph of current function
```
#### Strings:
```r
iz                     # list strings
iz~flag                # filter (grep-like)
axt @ str.xxx          # xrefs to a string
```
#### Xrefs:
```r
axt @ 0xADDR            # xrefs to address
afx                     # call xrefs from current function
afc                     # callgraph (text)
```
#### Search:
```r
/ hello                # search string in binary
/x 90e8                 # search hex bytes
/wi flag                # search wide string
```
#### Imports / Symbols :
```r
ii                     # imports
ii~puts                # filter imports
is                     # symbols
iE                     # exports
```
#### Sections / entry :
```r
iI                     # binary info
iS                     # sections
ie                     # entrypoint
```
#### Debugging (r2 -d):

```r
db 0xADDR              # breakpoint
db                     # list bps
dc                     # continue
ds                     # step
dso                    # step over
dr                     # registers
dr rip=0xADDR          # set reg
pxr 64 @ rsp           # stack peek
```
#### Patching:
```r
oo+                    # reopen in write mode (or start r2 -w)
wx 9090                # write bytes (NOP NOP) at current
wa jmp 0xADDR          # assemble + write instruction
wq                     # write/quit (if in -w)
```
#### File-offset patching:
```r
e io.va=false
s 0xOFF
px 32
wx ...                 # patch bytes at file offset
```
